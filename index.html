We wrote a server status page for a community web site/forum for a popular game called Knight Online. We developed a script that would send packets down to the official servers and the servers would respond with reply packet data. We then parsed this data for specific information that we needed.

We got information such as: how many players were online, which servers were online, how many users could the servers potentially hold, and much more.

For this project, we used Perl for its incredible parsing ability and easy TCP/IP socket communication setup. We created the frontend using the Mojolicious framework, HTML5, and Javascript. Then we set up a linux server with Perl installed and ran Mojolicious.
